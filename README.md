# Population_Analysis_Yucatan

Demographic and population analysis of Yucatán using statistical methods, data visualization, and exploratory data analysis.

---
This repository contains a demographic and statistical analysis of the population of Yucatán, Mexico. The project explores population dynamics, demographic trends, and regional patterns using data analysis, visualization techniques, and statistical methods implemented in Python.

The main objective of this project is to study demographic behavior in Yucatán through exploratory data analysis and statistical interpretation.
The analysis includes:

- Population proration from INEGI Census data (2010 & 2020)
- Census correction using the 1/16 method
- Exponential population growth modeling
- Specific mortality and fertility rates
- Life tables (Tablas de mortalidad)
- Global Fertility Rate and Mean Age of Fertility
- Migration analysis
- Population projections
